# Candle exploration sandbox

This folder is a scratchpad for trying ideas (notebooks and small binaries). Content here is experimental and may change or be removed. Nothing in this directory implies upstream Candle support until merged there.

Notes
- Prefer small, focused examples over large frameworks.
- Clean up stale experiments regularly.
- If a demo matures, split it out and propose it upstream with tests and docs.
